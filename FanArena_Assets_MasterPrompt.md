# FanArena_Assets_MasterPrompt.md

Master prompt for generating a cohesive asset pack for **FanArena**, a neutral offline sports-fan companion app built in Flutter.

## Global Style Rules

Create a modern, neutral, energetic sports-inspired illustration set for a cross-platform mobile app.

### Style
- Clean semi-flat illustration style with subtle depth
- Modern app-friendly visuals, not photorealistic
- Not cartoonish for children
- No gambling, betting slips, slot-machine, casino, or sportsbook references
- No direct references to real clubs, leagues, or copyrighted team logos
- Friendly, premium, organized, sporty

### Mood
- Exciting but clean
- Motivating and fan-oriented
- Balanced energy rather than chaotic action
- Suitable for both iOS and Android

### Palette Suggestions
- Primary blue: `#1E6BFF`
- Deep blue: `#1449B5`
- Accent orange: `#FF7A00`
- Success green: `#18A957`
- Soft background gray: `#F4F6F8`
- Dark navy: `#0F1720`
- White: `#FFFFFF`
- Neutral text gray: `#667085`

### Lighting & Rendering
- Soft gradients only
- Very light shadows
- Minimal glow
- Clean edges and readable silhouettes

### Composition Rules
- Center-focused subjects
- 10% safe padding from all edges
- Clear negative space for UI overlays
- Works in both Light and Dark themes
- No tiny details that disappear on phone screens

### Output Rules
- PNG format
- High resolution
- Transparent background where specified
- Solid or softly textured background only where specified
- No text unless explicitly requested
- No logos, no trademarks, no shirt sponsors, no branded stadiums

## Master Generation Prompt

Create a cohesive premium asset pack for a mobile app called **FanArena**, a neutral offline sports-fan organizer. The style should be modern semi-flat with subtle depth, clean edges, soft gradients, and a sporty but non-aggressive visual language. Use a restrained palette built around blue, dark navy, orange, green, white, and light neutral gray. Avoid all real-world branding, league identity, club logos, jersey sponsors, and any gambling or betting associations. Keep compositions centered, readable on mobile screens, and safe for UI overlays with about 10% padding. All assets must feel like part of one product family. Use transparent backgrounds for in-app empty states and decorative assets, and soft solid backgrounds for onboarding illustrations. Do not include any text in the images.
